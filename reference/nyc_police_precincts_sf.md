# New York City Police Precincts

Simple feature collection of NYC Police Precincts. EPSG:2263, NAD83 /
New York Long Island (ftUS).

## Usage

``` r
nyc_police_precincts_sf
```

## Format

### `nyc_police_precincts_sf`

A simple feature collection with 78 rows and 4 columns:

- precinct:

  Police precinct number

- shape_leng:

  Shape length

- shape_area:

  Shape area

- geometry:

  Multipolygon

## Source

<https://www.nyc.gov/content/planning/pages/resources#datasets>

## Details

Boundaries of NYC Police Precincts. Produced by NYC Planning Department.
Release: 25C, August 2025.
