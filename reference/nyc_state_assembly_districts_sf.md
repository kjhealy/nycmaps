# New York State Assembly Districts

Simple feature collection of New York State Assembly Districts within
NYC. EPSG:2263, NAD83 / New York Long Island (ftUS).

## Usage

``` r
nyc_state_assembly_districts_sf
```

## Format

### `nyc_state_assembly_districts_sf`

A simple feature collection with 65 rows and 4 columns:

- assem_dist:

  State Assembly district number

- shape_leng:

  Shape length

- shape_area:

  Shape area

- geometry:

  Multipolygon

## Source

<https://www.nyc.gov/content/planning/pages/resources#datasets>

## Details

Boundaries of New York State Assembly Districts within NYC. Produced by
NYC Planning Department. Release: 25C, August 2025.
