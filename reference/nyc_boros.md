# New York City borough names

Borough and county names and code.

## Usage

``` r
nyc_boros
```

## Format

### `nyc_boros`

A data frame with 5 rows and 5 columns:

- boro_code:

  City borough code. (Numeric)

- boro_name:

  Borough name

- county_name:

  County_Name County

- short_county_name:

  County_Name

- long_county_name:

  County_Name County, New York
