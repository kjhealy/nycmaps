# New York City Election Precincts

Simple feature collection of NYC Election Precincts. EPSG:2263, NAD83 /
New York Long Island (ftUS).

## Usage

``` r
nyc_election_precincts_sf
```

## Format

### `nyc_election_precincts_sf`

A simple feature collection with 4264 rows and 4 columns:

- elect_dist:

  Election district identifier

- shape_leng:

  Shape length

- shape_area:

  Shape area

- geometry:

  Multipolygon

## Source

<https://www.nyc.gov/content/planning/pages/resources#datasets>

## Details

Boundaries of NYC Election Districts (also known as Election Precincts).
Produced by NYC Planning Department. Release: 25C, August 2025.
